## A propos de l'application
 Une application web permettant à une structure spécialisée dans la vente de burger de gerer ses ventes de burger

## Fonctionnalités
 -  Espace d'administration  
 -  Ajout
 -  Modification
 -  Suppression

## Technologies

  -  FrontEnd ( Bootstrap 5 )
  -  BackEnd ( PHP 8.2 )
  -  BD ( MySQL )
